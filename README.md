
# Answers of the questions:


 
    - 1️⃣ What is the difference between var, let, and const?
         ans: JavaScript uses var, let, and const to declare variables, which differ primarily in scope and reassign ability. var is old-fashioned, function-scoped, and can be reassigned. let is block-scoped (only inside {}) and allows updating, while const is block-scoped, immutable, and is preferred for constant values.

    - 2️⃣ What is the spread operator (...)?
         ans: The spread operator (...) in JavaScript is a feature that allows you to take an array, object, or string and "spread" its individual items or properties out into a new location.

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
         ans:the main difference lies in their purpose and return value: map() and filter() both return a new array, while forEach() returns undefined and is used for side effects.
    - 4️⃣ What is an arrow function?
         ans: An arrow function is a compact, modern way to write functions in JavaScript
    - 5️⃣ What are template literals?
         ans: Template literals are a JavaScript feature that allows for easy string formatting, string interpolation, and multi-line string creation.
